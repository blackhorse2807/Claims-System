# Health Insurance Claims Processing System

Automated multi-agent pipeline for processing employee health insurance claims against policy rules.

## Quick Start

### Prerequisites

- Node.js 18+
- An Anthropic API key (only needed for real file upload parsing)

### Setup

```bash
# 1. Clone and enter the project
cd claims-system

# 2. Install backend dependencies
cd backend
npm install

# 3. Add your API key (optional — tests work without it)
# Edit backend/.env and set:
# ANTHROPIC_API_KEY=sk-ant-...

# 4. Start the backend
npm start

# 5. In a new terminal, install and start the frontend
cd ../frontend
npm install
npm run dev
```

Open http://localhost:5173

### Run all 12 test cases

```bash
cd backend
npm test
```

## API

### POST /api/claims

Submit a claim for processing.

**Body (form-encoded or JSON):**

- `member_id` — e.g. "EMP001"
- `claim_category` — CONSULTATION | DIAGNOSTIC | PHARMACY | DENTAL | VISION | ALTERNATIVE_MEDICINE
- `treatment_date` — YYYY-MM-DD
- `claimed_amount` — number
- `hospital_name` — string (used for network discount detection)
- `ytd_claims_amount` — amount already claimed this year
- `claims_history` — JSON array of previous claims
- `documents` — JSON array of document objects with content field (test mode)
- `files` — multipart file uploads (real documents)

**Response:**

```json
{
  "claim_id": "uuid",
  "decision": "APPROVED",
  "approved_amount": 1350,
  "claimed_amount": 1500,
  "confidence_score": 0.92,
  "reasons": ["10% copay applied"],
  "rejection_reasons": [],
  "line_items": [],
  "trace": [],
  "component_failures": []
}
```
